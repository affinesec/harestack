# Harestack

A charming browser physics game about stacking hares and making questionable structural decisions.

## Play locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Production build

```bash
npm run build
```

The static site is generated in `dist/` and can be deployed directly to Cloudflare Pages, Netlify, Vercel, or any static host.

## Included in the prototype

- Daily deterministic stack and an endless mode
- Six hare types with distinct size, mass, friction, and bounce
- Mouse, touch, and keyboard controls
- Increasing wind pressure, carrots, sound, particles, and result cards
- Local personal-best tracking and native share/clipboard support
